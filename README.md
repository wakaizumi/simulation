# Simulation
研究で作成したコードの一部(提出用)．人の動きを模倣した人流シミュレータ内で混雑に関する経路推薦を行っている．  
1. def1.py データ，マップ操作に関する関数
2. def2_2.py 探索アルゴリズムに関する関数
3. def3.py 描画に関する関数
4. agent4.py マップ上を動くエージェントのクラス
5. シミュレーション　東京駅周辺マップ内でエージェントを動かす実験を行っている (Jupyter Notebook)

使用データ　国土交通省「歩行者移動支援サービスに関するデータサイト」https://www.hokoukukan.go.jp/top.html
